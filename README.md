# 🌌 PISAG – Plataforma de Aprendizaje Astronómico

PISAG es una aplicación web interactiva para aprender sobre astronomía mediante noticias, quizzes y un foro.  
Está dirigida a estudiantes, aficionados al espacio y autodidactas mayores de 15 años.

---

## 🚀 Funcionalidades

- 📰 Noticias espaciales actualizadas (NASA, ESA, Space.com, etc.)
- 🧠 Quizzes para poner a prueba tus conocimientos
- 💬 Información acerca de temas astronomicos
- 🛠️ Inicios de sesión y registro
- 🚀 Galeria por medio de la API de la nasa (https://github.com/nasa/apod-api)
